Description [src]

Create an array of all available values of enumeration, as strings.

Available since:0.8.0

Parameters

n_values guint*
 

Placeholder for the number of values.
 The argument will be set by the function.
error GError **
  The return location for a GError*, or NULL.

Return value

Returns: An array of utf8
 

An newly created array of const strings, which must freed after use using g_free, NULL on error.
 The length of the array is in the n_values argument.
 The caller of the method takes ownership of the data container, but not the data inside it.
 Each element is a NUL terminated UTF-8 string.
